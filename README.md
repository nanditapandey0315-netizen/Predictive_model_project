# Titanic Survival Prediction using Machine Learning

## Project Overview
This project predicts whether a passenger survived the Titanic disaster using machine learning classification algorithms.

## Objectives
- Clean and preprocess the dataset.
- Train machine learning models.
- Compare model performance.
- Evaluate prediction accuracy.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Dataset
Titanic Dataset

## Algorithms Used
- Decision Tree Classifier
- Random Forest Classifier

## Results
- Decision Tree Accuracy: 73.18%
- Random Forest Accuracy: 79.33%

## Outcome
Built and evaluated machine learning models to predict passenger survival. Random Forest performed better than Decision Tree in terms of accuracy.
